## 🧩 Pixona

**'픽소나(Pixona)'** 는 사용자가 입력한 **이미지**를 바탕으로, **생성형 AI**를 통해
**본인만의 ‘픽셀 아트 아바타’를 생성**하는 서비스 입니다.

본인의 셀프 카메라 사진, 전신 사진, 심지어는 본인이 그린 그림까지.
픽소나(Pixona)는 특정한 사진 유형에 구애 받지 않습니다.
본인만의 사진 한 장으로, 원하는 특징을 갖춘 아바타를 생성 해 보세요!
